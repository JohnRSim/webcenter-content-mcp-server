Claude Desktop Configuration

Add this to your Claude Desktop configuration file:

{
  "mcpServers": {
    "webcenter-content": {
      "command": "C:\\path\\to\\WebCenter Content MCP Server.exe",
      "env": {
        "WCC_BASE_URL": "https://your-server.com/documents/wcc/api/v1.1",
        "WCC_USER": "your-username",
        "WCC_PASSWORD": "your-password"
      }
    }
  }
}
